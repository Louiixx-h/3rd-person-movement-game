# Third Person Game

Este protótipo de jogo em terceira pessoa tem como objetivo recriar as mecânicas de movimentos usadas em jogos de terceira pessoa.

### 🏃‍♀️Movimentos Básico

- [x] Andar
- [x] Correr
- [x] Pular
- [x] Deslizar
- [x] Camera Orbital
- [x] Direção do personagem segue o da câmera
